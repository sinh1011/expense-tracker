# 💰 Expense Tracker

Ứng dụng quản lý chi tiêu cá nhân giúp bạn theo dõi thu nhập và chi tiêu một cách dễ dàng.

## 🚀 Demo

**🔗 [Xem live tại đây](https://sinh1011.github.io/expense-tracker/)**

## ✨ Tính năng

- Thêm giao dịch thu nhập và chi tiêu
- Hiển thị tổng số dư, tổng thu nhập, tổng chi tiêu
- Lưu dữ liệu vào localStorage (không mất dữ liệu khi reload)
- Xóa từng giao dịch hoặc xóa tất cả
- Định dạng tiền tệ Việt Nam (VNĐ)
- Giao diện responsive, hiện đại

## 🛠️ Công nghệ

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+, localStorage API)
